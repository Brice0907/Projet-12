# Installation

1. Cloner le repository :
 - `git clone https://github.com/Brice0907/Projet-12`
2. `cd Front-end`
3. Installation des dépendances :
 - `npm install`
4. Lancer le Front :
 - `npm run dev`

# Description 

- Faire la totalité du Front du projet, lisible sur les écrans d’au moins 1024 par 780 pixels
- Utiliser D3 ou Recharts pour générer les grapgiques
- Utiliser une API en faisant des calls HTTP sur le backend NodeJS fourni en utilisant Fetch ou Axios

# Compétence évaluées

- Assurer la qualité des données d'une application
- Développer des éléments graphiques avancés à l'aide de bibliothèques JavaScript
- Interagir avec un service Web
